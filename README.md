## 📌 Project Overview

The **Diwali Sales Analysis** project is an **Exploratory Data Analysis (EDA)** project developed using **Python and Jupyter Notebook**. The primary objective of this project is to analyze customer purchasing behavior during the Diwali festive season and derive actionable business insights.

The project demonstrates an end-to-end data analysis workflow, including:

✔️ Data Cleaning and Preprocessing  
✔️ Exploratory Data Analysis (EDA)  
✔️ Data Visualization  
✔️ Customer Segmentation  
✔️ Business Insight Generation  
✔️ Data Storytelling

---

## 🎯 Business Problem

During festive seasons, businesses generate massive amounts of sales data. However, without proper analysis, it becomes difficult to answer questions such as:

- Who are the most valuable customers?
- Which states generate the highest sales?
- Which products are highly demanded?
- Which occupations contribute the most revenue?
- Which customer segments should be targeted for future marketing campaigns?

This project aims to answer these business questions through data-driven analysis.

---

## 📂 Dataset Information

The dataset contains transactional records of customers during the Diwali festival season, including:

- User ID
- Gender
- Age Group
- Marital Status
- State
- Occupation
- Product Category
- Product ID
- Orders
- Purchase Amount

## 🔄 Project Workflow

### 1️⃣ Data Collection
- Imported the Diwali Sales dataset into Jupyter Notebook.

### 2️⃣ Data Cleaning & Preprocessing
- Removed unnecessary columns.
- Checked for missing values.
- Removed null values.
- Converted data types.
- Renamed columns where necessary.

### 3️⃣ Exploratory Data Analysis (EDA)
Performed analysis based on:

- Gender
- Age Group
- Marital Status
- State
- Occupation
- Product Category
- Product Orders
- Purchase Amount

### 4️⃣ Data Visualization
Created multiple visualizations using:

- Bar Charts
- Count Plots
- Distribution Plots
- Customer Segmentation Charts

### 5️⃣ Business Insight Generation
Derived meaningful insights and recommendations for business decision-making.

# 📊 Key Insights

### 👩 Gender Analysis
- Female customers contributed more to overall sales than male customers.

### 👥 Age Group Analysis
- Customers aged **26–35 years** represented the largest purchasing segment.

### 💍 Marital Status Analysis
- Married women showed higher purchasing behavior.

### 📍 State Analysis
Top-performing states included:

🥇 Uttar Pradesh  
🥈 Maharashtra  
🥉 Karnataka

### 💼 Occupation Analysis
Customers working in:

- IT
- Healthcare
- Aviation

generated significant revenue.

### 🛍️ Product Category Analysis
The most popular product categories included:

- Food
- Clothing & Apparel
- Electronics

---

# 💡 Business Recommendations

✅ Target marketing campaigns toward women aged 26–35 years.

✅ Increase promotional efforts in high-performing states.

✅ Maintain inventory for high-demand products.

✅ Introduce loyalty and retention programs.

✅ Use customer segmentation for personalized marketing.
